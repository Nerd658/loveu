## 2024-05-24 - Custom Focus Outlines for Circular and Complex Shapes
**Learning:** Default browser focus rings (`outline: auto`) on custom interactive elements (like `canvas` elements acting as buttons, e.g. `#constCanvas` which is visually circular, or complex CSS shapes like the envelope `#envelope`) often result in visually confusing rectangular outlines that clash with the element's intended shape and lack adequate contrast on dark, thematic backgrounds.
**Action:** When creating non-rectangular interactive elements (especially `<canvas>` or `div` with `role="button"`), explicitly override the `:focus-visible` state. Use `border-radius` to match the element's visual shape (e.g., `border-radius: 50%` for circles) and apply custom `outline` or `box-shadow` properties using existing design system tokens (like `--gold`) to ensure accessible, high-contrast, and aesthetically cohesive focus indicators.
